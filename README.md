# 🚀 FirstLayer Releases
Die offiziellen Releases von FirstLayer – deinem smarten Chat-Layer für Streams.
Hier findest du die aktuellen Builds, ältere Versionen sowie alle Änderungen (Changelog), um immer auf dem neuesten Stand zu bleiben.

## 📥 Download
- Lade dir einfach die neueste Version aus dem Bereich Releases herunter:
- ➡️ rechts auf → Releases → neueste Version auswählen → .exe herunterladen und starten

💡 **Hinweis:**
- Windows kann beim ersten Start eine Sicherheitswarnung anzeigen („Unbekannter Herausgeber“).
- Das ist normal bei neuen Apps ohne verbreitetes Code Signing.

## ⚡ Installation**
- .exe herunterladen
- Datei ausführen
- Setup durchlaufen
- FirstLayer starten
- Keine zusätzlichen Abhängigkeiten notwendig.

## 🔄 Updates
- Updates passieren bereits als Auto-update über die App. 
- Schaut ab und zu mal in den System-Tab

## 🧾 Changelog
- Jede Version enthält ein eigenes Changelog direkt im Release.
- **Typische Inhalte:**
- ✨ Neue Features
- 🐛 Bugfixes
- ⚡ Performance Improvements
- 🧪 Experimentelle Features

## 🧠 Was ist FirstLayer?
- FirstLayer ist mehr als ein Chat-Overlay:
- **🎮 In-Game Overlay** – Twitch-Chat direkt über deinem Spiel
- **🧠 Smart Chat** – erkennt Muster, merged Spam („GG ×42“)
- **📋 Activity Feed** – wichtige Events separat
- **🤖 Pulse AI** – erkennt Stimmung & Topics live
- **👥 Viewer Insights** – verstehe deine Community besser
- **👉 Ziel:** Weniger Chaos im Chat, mehr Fokus auf deinen Stream
